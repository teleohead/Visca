# Visca

Visca is a Scala-based compiler project ported from the original Java-based VC (a Variant of C) compiler.

## Getting Started
**Prerequisites**:
- JDK 21+
- Scala 3.3+
- sbt 1.9+

## Usage
To compile, run
```
sbt compile
```
